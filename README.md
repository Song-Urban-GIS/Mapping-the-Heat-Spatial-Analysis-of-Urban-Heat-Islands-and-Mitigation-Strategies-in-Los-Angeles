# Mapping the Heat: Spatial Analysis of Urban Heat Islands and Mitigation Strategies in Los Angeles
GIS-based investigation into Urban Heat Island (UHI) intensity in LA County. This project correlates Land Surface Temperature (LST) with tree canopy coverage and socioeconomic vulnerability to propose targeted, data-driven urban cooling interventions.
